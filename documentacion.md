# Documentación técnica

# Estructura del proyecto
El proyecto está organizado en diferentes clases que representan las entidades del videoclub.

# Clases principales

# Cliente
Representa a un cliente del videoclub.

Atributos:
- id
- nombre

Métodos:
- alquilar()
- devolver()

# Pelicula
Representa una película disponible.

Atributos:
- titulo
- genero

# Alquiler
Relaciona un cliente con una película.

Atributos:
- cliente
- pelicula
- fecha

# Funcionamiento
El sistema permite registrar clientes, añadir películas y gestionar alquileres.
